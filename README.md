# Backend

## :computer: Run it!

```bash
# enter the folder
cd backend

# Install the dependencies
yarn install

# Run migrations to your database
yarn knex migrate:latest

# Run the backend server
yarn start

```
# Web

## :computer: Run it

```bash
# enter the folder
cd web

# Install the dependencies
yarn install

# Start the project
yarn start

```
